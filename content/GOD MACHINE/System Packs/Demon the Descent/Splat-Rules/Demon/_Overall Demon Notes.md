# General Advantages
## Total Control
Demons have total control over their bodily actions.
Additionally, it is IMPOSSIBLE to tell if a demon is lying based on physical tells.
It is also impossible to tell if a demon is lying as long as it is true for ONE cover.
# Natural Aptitude
Demons have the [[Eidetic Memory]] merit as well as being fluent in every native language currently in use. They do not understand non-native languages, but they CAN learn it. They can speak First Tongue, but not High Speech.


# Going Loud
Demons can go loud, which lasts for a scene. Going Loud does the following
- Sets Cover to 0, see [[Cover]] for more.
- Assumes demonic form automatically
- Primum is set at 10
	- No permanent glitches, but does gain a transient glitch
- Aether Pool is filled (remember Primium 10)
- Gains access to **every fucking** Embed the incarnation gives affinity for and can use any Exploit.
